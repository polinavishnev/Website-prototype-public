# Capstone Write-up - Discearn Web App

### Introduction

This project is part of the ongoing development of the Discearn Web App, a project dedicated to improving the online learning experience. The application is designed to provide effective methods for reviewing and synthesizing information from diverse sources. Following an Agile development approach, the project is in its early stages, with ten user interviews completed. The plan is to continue with another cycle of interviews, followed by further development, ensuring that the product aligns with user needs and preferences.


**Functional Requirements**:

- [X] The web app can generate questions.
- [X] The web app can provide feedback, including feedback on user responses and the sources used for feedback.
- [ ] The web app displays questions based on an optimal schedule for retention and review.
- [ ] The web app allows users to specify their preferences regarding the content they wish to be quizzed on each day.

**Nonfunctional Requirements**

- [ ] The web app must offer an intuitive and user-friendly navigation experience.
- [ ] It should display OpenAI's output as it generates, providing a seamless and efficient user experience.



**Developing a Product**

Presently, the app comprises a React application hosted on Render and has the capability to send requests to OpenAI.
